## 二、段落
1. 无序列表
> "*" + " " + 列表内容

2. 首行缩进
在段落首行缩进的位置，先写下面这个代码(包括分号)，然后再继续写后面的内容就可以了。这个代码写一次是空一格，两个就是空两格。

```
&emsp;
```

3. 居中
```Bash
# <center>一、习惯C++
```

4. 换行
```Bash
<br />
```

5. 表格
使用jekyll时，如果选择kramdown作为引擎，表示表格时，如果使用表头，则表格前两行的结尾需要两个空格作为换行
